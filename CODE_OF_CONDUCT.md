# Code of Conduct

CampusForge is built for respectful, inclusive, and constructive collaboration.

## Standards

Expected behavior:

- Treat peers, maintainers, instructors, and administrators with respect.
- Keep discussions focused on learning, building, and improving projects.
- Attribute resources and code correctly.
- Report security or safety issues through the appropriate channel.

Unacceptable behavior:

- Harassment, discrimination, intimidation, or personal attacks.
- Sharing invitations or internal materials without permission.
- Misrepresenting identity, role, affiliation, or contribution history.
- Publishing private student data, credentials, or school-only materials.

## Enforcement

Administrators may remove content, suspend access, request clarification, or revoke membership. All moderation actions should be logged in the audit system.
