# 04. 设计系统

## 视觉关键词

- 深色校园夜景
- 星图 / 网络线条
- 紫蓝青渐变
- 细边框发光
- 低信息密度
- 卡片式结构
- 科技感但克制

## 色彩令牌

| Token        | Value     | 用途              |
| ------------ | --------- | ----------------- |
| `--bg-950`   | `#050915` | 页面背景          |
| `--bg-900`   | `#07101f` | 主卡片背景        |
| `--bg-800`   | `#0c172a` | 次级卡片背景      |
| `--text-100` | `#f8fbff` | 主文字            |
| `--text-300` | `#b6c3d9` | 正文              |
| `--text-500` | `#71809a` | 辅助文字          |
| `--purple`   | `#7c4dff` | 品牌重点          |
| `--blue`     | `#2f80ff` | 交互重点          |
| `--cyan`     | `#12d7e8` | 强 CTA / 成功状态 |
| `--danger`   | `#ff5f7d` | 拒绝 / 风险       |
| `--success`  | `#21d6a2` | 通过 / 成功       |

## 布局规则

- 头部导航固定视觉层级，内容区最大宽度 1280px。
- 首页采用左文案 + 右邀请卡 + 下方三卡预览。
- 管理后台采用左侧栏 + 主表格 + 右侧详情抽屉。
- 页面标题不超过两行，副文案控制在 1–2 行。
- 每屏卡片数量尽量控制在 3–6 个。

## 组件规范

### Glass Card

- 背景：半透明深色。
- 边框：`1px solid rgba(125, 150, 255, 0.18)`。
- Hover：轻微边框和阴影增强。

### Primary Button

- 渐变：紫 → 蓝 → 青。
- 只用于核心动作：Request Invitation、Submit、Approve。

### Tags

- 小圆角胶囊。
- 资源/课程标签使用低饱和背景，避免喧宾夺主。
