# 10. 管理员操作手册

## 审核申请

1. 进入 Admin → Application Review。
2. 查看申请人基本信息、院系、兴趣与加入理由。
3. 根据邀请规则判断：
   - 通过：身份清晰、动机合理、符合社区规则。
   - 要求补充信息：资料不足或表述不清。
   - 拒绝：不符合身份或安全要求。
4. 所有操作写入审计日志。

## 管理资源

- 确认资源来源、许可证和适用对象。
- 使用清晰标题和短描述。
- 过期资源标记为 Archived。

## 管理制度

- 制度文档发布前需要管理员评审。
- 更新后记录版本、日期和变更摘要。
- 关键制度需要在首页或资源中心显著展示。
